## 🚨 우리가 풀려는 문제

**한국 유기견 11만 마리/년**, 늘어나는 추세.

근본 원인은 단순합니다.

- **① 안일한 입양** — "귀여워서" 데려왔다가 현실(짖음·털·의료비·문제 행동)에 부딪힘
- **② 훈련 부족** — 짖음·분리불안 같은 문제 행동이 유기의 가장 큰 원인
- **③ 동반 가능 공간 협소** — 카페·식당·공원 대부분 거부 → 보호자·강아지 모두 사회적 고립

> 기존 입양 매칭 서비스는 "어떤 견종이 맞나"만 묻습니다. **"당신, 정말 준비됐나요?"** 는 묻지 않습니다.

## 💡 우리의 답 — 4단계 흐름

### ① 입양 준비도 진단 (후킹) — 인지부터 바꿉니다

```
"작은 강아지가 키우기 쉬울 거라 생각하세요?"
"이사 갈 집이 펫 금지면 다른 집 찾을 여유 있으세요?"
"귀여운 강아지가 물면 어떨 것 같으세요?"
```

→ **데이터 기반 진실**을 알려줍니다. (말티즈는 사실 슬개골 위험·짖음 ↑)
→ 단순 매칭이 아니라 **"진지함의 자가 진단"**.

### ② 라이프스타일 매칭 — 통과한 사람만 진짜 매칭

후킹 통과자에게만 견종 추천. 비용·산책·훈련 + 외모 + 한국 시장 인기까지 통합 매칭.

### ③ 입양 후 AI 훈련 ⭐ 핵심 가치

- Claude·Twelve Labs API 활용한 영상 분석 훈련
- 견종 성격에 맞춘 개인화 코칭
- 훈련 진도 측정 + 인증서 발급

### ④ 착한개 인증 제도 ⭐ 차별화 핵심

> **"인증받은 강아지는 어디든 환영받습니다."**

- 훈련 완료 강아지에게 디지털 인증서
- 인증 강아지 전용 동반 가능 매장·공원 네트워크
- 보호자·강아지의 사회 참여 권리 확장

## 🌱 생태계 효과

```
[준비된 보호자] → [잘 훈련된 강아지] → [사회 어디든 환영]
      ↑                                       ↓
[유기 위험 ↓]                       [반려문화 격상]
```

**4단계가 서로 강화**합니다. 한 단계만 떼어내면 안 보입니다.
