## 3. 한국 주거 환경 특성

### 아파트 문화 반영

한국 반려인의 약 **60% 이상이 아파트 거주**. 이에 따라 아래 기준이 중요:

| 기준 | AKC 데이터 활용 | 한국 특화 포인트 |
|------|----------------|----------------|
| 짖음 수준 | `barking_level` | 층간소음 민감 → 점수 1~2 선호 |
| 공간 적응력 | `adaptability_level` | 높을수록 좋음 |
| 운동 필요량 | `energy_level` | 산책 가능 환경에 따라 달라짐 |
| 크기 | `size` | 소형 선호 (한국 아파트 평균 25~34평) |

```
한국 아파트 적합 추천 기준 (서비스 필터 예시):
- size: Small 또는 Medium
- energy_level: 1~3
- barking_level: 1~2
- adaptability_level: 3 이상
```

### 한국에서 견종 선택 시 현실적 고려사항

| 항목 | 설명 |
|------|------|
| **미용비** | 털이 길거나 곱슬인 견종은 정기 미용비 발생 (말티즈, 푸들 등) |
| **산책 환경** | 소형견은 아파트 단지 내 산책 가능, 대형견은 공원 필요 |
| **펫 케어 비용** | 한국은 수의비가 높은 편 → 건강 문제 많은 견종 주의 |
| **계절** | 한국의 덥고 습한 여름 → 단두종(프렌치불독, 퍼그 등) 열사병 주의 |

## 5. 한국 반려동물 등록 의무

- **법적 의무**: 2개월령 이상 반려견은 내장형 칩 또는 외장형 인식표로 등록 의무
- **미등록 시**: 최대 60만원 과태료
- **관련 서비스**: 동물보호관리시스템 (www.animal.go.kr)

## 6. 서비스 UX 권장사항 (한국 맥락)

### 경고 표시 권장 케이스

| 상황 | 표시 내용 |
|------|----------|
| `is_dangerous_breed_kr: true` | 🚨 "이 견종은 동물보호법상 맹견입니다. 외출 시 입마개·목줄 착용이 법적 의무입니다." |
| `barking_level >= 4` | 🔔 "짖음이 많은 편입니다. 아파트 환경에서는 충분한 훈련이 필요합니다." |
| 단두종인 경우 | 🌡️ "한국의 무더운 여름에 열사병 위험이 있습니다. 냉방 환경을 항상 유지해야 합니다." |
| `energy_level >= 4` AND `size >= Large` | 🏃 "충분한 운동 공간이 필요합니다. 마당이 있는 주거 환경을 권장합니다." |
| `good_with_young_children <= 2` | 👶 "어린 아이와 함께하기에 주의가 필요한 견종입니다." |

### 필터 추천 항목 (한국 사용자 맞춤)

```
자주 묻는 질문 → 필터 연결:

Q: 아파트에서 키워도 될까요?    → recommended_for.apartment
Q: 아이가 있어도 안전한가요?    → child_safety_score, children_under_5
Q: 털 빠짐이 적은 개를 원해요   → shedding_level (낮은 순)
Q: 훈련이 쉬운 개를 원해요      → trainability_level (높은 순)
Q: 맹견은 아닌가요?             → is_dangerous_breed_kr
Q: 자주 짖지 않는 개 원해요     → barking_level (낮은 순)
Q: 혼자 있는 시간이 많아요      → adaptability_level, energy_level
```
