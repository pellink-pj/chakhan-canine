Tiny pill label for trait tags, the 착한개 certification mark, and Korea popularity ranks. No emoji — use the tone color to carry meaning.

```jsx
<Badge tone="gold" solid>한국 인기 29위</Badge>
<Badge tone="hope">훈련 가능</Badge>
<Badge tone="good">털 빠짐 적음</Badge>
<Badge tone="info">AI 훈련</Badge>
```

Tones map to the brand tone-trio: `good` green, `hope` amber, `caution` coral-red, plus `brand`, `info` (electric blue), `gold` (ranks), `neutral`. `solid` fills it; default is tinted. An optional `icon` (e.g. a Lucide node) is supported but unused by default.
