The heart of MungMate — turns a breed trait into a hope-first insight. Always frame negatives as overcomeable (`tone="hope"`), never a bare warning. The brand uses no emoji; tone color + the left accent bar carry the meaning.

```jsx
<TraitCard
  tone="hope"
  title="원래 잘 짖지만, 훈련으로 컨트롤 가능해요"
  text="이 견종은 타고난 경계심으로 짖음이 많은 편이에요. 하지만 학습 의지가 강해 꾸준한 훈련으로 컨트롤할 수 있어요."
/>
```

Tones: `good` (green), `hope` (amber — the default and most-used), `caution` (coral-red, use sparingly). The tone label is auto-set in Korean. An optional `icon` (e.g. a Lucide node) may be passed but is omitted by default.
