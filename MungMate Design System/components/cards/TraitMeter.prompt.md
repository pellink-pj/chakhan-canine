A single AKC-style 1–5 trait row for breed profiles.

```jsx
<TraitMeter label="짖음" score={5} scaleWord="매우 높음" tone="hope" />
<TraitMeter label="훈련 가능성" score={5} scaleWord="매우 쉬움" tone="good" />
<TraitMeter label="털 빠짐" score={3} scaleWord="보통" />
```

Stack several in a column. Use `tone` to color-code: `hope` for a demanding-but-trainable trait, `caution` for a genuine concern, `good` for a plus.
